# Community Depression Intervention in Miami-Dade County

## Overview
This project presents a faith-based community mental health intervention designed to address depression in Miami-Dade County.

## Focus
- Depression in Miami-Dade County
- Community mental health disparities
- Faith-based support programs
- Mental health access and stigma
- PHQ-9 as an evaluation tool
- Long-term intervention outcomes

## Purpose
The purpose of this project is to propose a culturally responsive and community-based intervention that improves access to mental health support and reduces depression-related disparities.

## Skills Demonstrated
- Community intervention planning
- Program design
- Outcome measurement
- Public mental health strategy
- Community partnership planning
- Health equity awareness
